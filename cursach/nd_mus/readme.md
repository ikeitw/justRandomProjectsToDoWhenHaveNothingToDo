# 🎵 Музыкальный магазин (Node.js + Express + Sequelize + GUI)

## Описание
Веб-приложение для управления музыкальным магазином с графическим интерфейсом в браузере.

## 🚀 Запуск проекта

### 1. Установите зависимости
```bash
npm install
````

### 2. Создайте базу данных PostgreSQL

```bash
psql -U postgres
CREATE DATABASE musicshopdb;
\q
```

### 3. Настройте подключение к базе данных

Создайте файл `.env` в корне проекта:

```env
DB_NAME=musicshopdb
DB_USER=postgres
DB_PASS=ваш_пароль
DB_HOST=localhost
```

### 4. Заполните базу данных тестовыми данными (опционально)

```bash
node seed.js
```

### 5. Запустите приложение

```bash
npm start
```

Перейдите в браузере по адресу: [http://localhost:3000](http://localhost:3000)

## 📁 Структура проекта

| Файл / Папка   | Назначение                   |
| -------------- | ---------------------------- |
| `app.js`       | Основной Express сервер GUI  |
| `controllers/` | Бизнес-логика                |
| `models/`      | Sequelize модели             |
| `views/`       | EJS страницы                 |
| `public/`      | Статические файлы (CSS/JS)   |
| `seed.js`      | Заполнение тестовыми данными |
| `.env`         | Переменные подключения БД    |
| `package.json` | Зависимости и скрипты        |
