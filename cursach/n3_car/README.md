
# 🚗 Система продажи автомобилей

Информационная система для управления продажей новых и подержанных автомобилей, соответствующая техническому заданию №39. Включает:

* каталог автомобилей,
* базу покупателей с предпочтениями,
* базу автосалонов,
* автоматический подбор автомобилей по запросам покупателей и наоборот.

---

## 📦 Состав проекта

```plaintext
n3_car/
├── backend/         # Node.js + Express + Sequelize (PostgreSQL)
├── frontend/        # React SPA
├── README.md        # эта инструкция
```

---

## 🚀 Быстрый старт

### 1. 🛠 Установка зависимостей

#### Backend

```bash
cd backend
npm install
```

#### Frontend

```bash
cd ../custom-frontend
npm install
```

---

### 2. 🗃 Настройка PostgreSQL

1. Убедитесь, что у вас установлен PostgreSQL и пользователь `postgres` с паролем `postgres` (или отредактируйте `.env`).
2. Создайте базу данных:

```sql
CREATE DATABASE car_sales;
```

---

### 3. 🌱 Наполнение базы тестовыми данными

```bash
cd backend
node seed.js
```

Это создаст таблицы и добавит тестовые автомобили, покупателей и салоны.

---

### 4. 🚀 Запуск серверов

#### Backend (порт 5000)

```bash
cd backend
node server.js
```

#### Frontend (порт 3000)

```bash
cd ../custom-frontend
npm start
```

---

## 🌐 Интерфейс

* Главная страница показывает список всех авто.
* Можно:

    * кликнуть на авто → показать покупателей, которым он подходит.
    * кликнуть на покупателя → показать подходящие автомобили.
    * кликнуть на автосалон → показать его автомобили.
* Есть формы для **добавления, редактирования, удаления** всего.
* Поддерживается фильтрация по параметрам (с пустыми значениями = любые).

```
DB_NAME=car_sales
DB_USER=postgres

```

## 🧠 Запросы, реализованные в API

* `GET /api/match/buyers-for-car/:id` — найти покупателей на конкретный авто
* `GET /api/match/cars-for-buyer/:id` — найти авто по пожеланиям покупателя
* `GET /api/cars` — все авто (с фильтрацией)
* `GET /api/buyers`, `POST`, `PUT`, `DELETE` — CRUD покупателей
* `GET /api/dealerships`, `POST`, `PUT`, `DELETE` — CRUD автосалонов
